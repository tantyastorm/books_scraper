# books_scraper

# Product Scraper

Scrapes product data (title, price, stock, image URL) from [books.toscrape.com](https://books.toscrape.com).

## Features
- Simple structure
- Clean code
- Outputs JSON

## Usage
```bash
pip install -r requirements.txt
python main.py
```

## Output
- Exports data to `output/products.json`
